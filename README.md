# QRCodeUIKit
一款让扫码变得简单的视图控制器。


##  安装 / Installation

方法一：`QRCodeUIKit` is available through CocoaPods. To install it, simply add the following line to your Podfile:

```
pod 'QRCodeUIKit'
```
